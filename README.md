# Hướng dẫn chạy project

## 1. Setup

1. Clone project về máy:
```bash
git clone https://github.com/cuong9565/ossd-chat-socket.git
cd ossd-chat-socket
```
2. Chạy file run.sh
```bash
bash run.sh
```
## 2. Chạy chương trình

1. Tạo 1 terminal chạy server.py
```bash
python3 server.py
```

2. Tạo 1 terminal chạy app.py
```bash
streamlit run app.py
```
